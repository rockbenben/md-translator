## Sync Direction

| Option | Description | Selected |
|--------|-------------|----------|
| 双向同步 | Scrolling either panel scrolls the other | ✓ |
| 单向同步 | Only one direction scrolls the other | |

**Notes:** More natural UX — user can scroll either panel.
