### 1. 添加移动端状态

```typescript
const [mobileActiveTab, setMobileActiveTab] = useState<'source' | 'translation'>('source');
```

### 2. 响应式布局结构
```tsx
<div className="h-full">
  {/* 操作栏: 翻译按钮等 — 始终显示 */}
  <Flex justify="space-between">...</Flex>

  {/* 桌面端: 分屏视图 — md 及以上显示 */}
  <div className="hidden md:block">
    <SplitPaneContainer leftPanel={leftPanel} rightPanel={rightPanel} />
  </div>

  {/* 移动端: 单面板 + 底部标签 — md 以下显示 */}
  <div className="block md:hidden">
    <Card className="h-full" title={mobileActiveTab === 'source' ? t("sourceArea") : t("translationResult")}>
      {/* 根据 tab 显示内容 */}
      {mobileActiveTab === 'source' ? leftPanelContent : rightPanelContent}
    </Card>
    {/* 底部 Tab Bar */}
    <Flex className="fixed bottom-0 left-0 right-0 bg-white dark:bg-gray-800 border-t">
      <Button onClick={() => setMobileActiveTab('source')}>原文</Button>
      <Button onClick={() => setMobileActiveTab('translation')}>译文</Button>
    </Flex>
  </div>
</div>
```

### 3. 移动端内容处理
移动端单独面板复用左侧/右侧面板的渲染逻辑，但去除 SplitPaneContainer 包装。

### 4. 预览模式支持
移动端同样支持预览模式 (isPreviewMode, previewContent)，只是没有分屏。
